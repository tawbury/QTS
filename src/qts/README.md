# QTS Core

## 개요
QTS 애플리케이션의 핵심 부트스트랩 및 설정 관리 모듈입니다.

## 주요 구성요소
- **Config Loader**: YAML 및 환경 변수 기반의 통합 설정 로딩 (`core/config`)
- **Bootstrap**: 애플리케이션 시작 시 필요한 초기화 작업 (로깅 설정, 디렉토리 생성 등)
- **Container**: 의존성 주입(DI) 컨테이너 (선택적 사용)

## 사용법
`src.qts` 패키지는 애플리케이션의 진입점(`main.py` 등)에서 가장 먼저 임포트되어야 하며, `config_loader`를 통해 시스템 전반의 설정을 초기화합니다.
